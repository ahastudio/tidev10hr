10Hour Live Project

말보다는 직접 코드로 보여드립니다. 10시간 동안 부스에서 앱 만들기 프로젝트!

Download this project as a .zip file Download this project as a tar.gz file

한국 타이타늄 사용자 모임의 10hr 프로젝트에 오신것을 환영합니다.

DevOn 한국 타이타늄 사용자 모임에서는 행사가 열리는 시간 10시간 동안 커뮤니티 회원들이 직접 앱을 만듭니다.

iOS, Android 네이티브 앱 게다가 서버까지 전부요?

네, 물론 입니다. Titanium에서 제공하는 Cloud 서비스까지 이용하여 서버를 만들고,
TiSDK를 이용하여 iOS, Android 네이티브 앱을 만듭니다.

어떤 앱? 각종 경품을 드리기 위한 앱!! 받고 싶다면 지금 다운!

저희가 준비한 사은품이 많아요!! 받고 싶다면 앱을 지금 다운 받아 로그인 하세요. 좋은 경품, 더 높은 당첨 확률을 원한다면 주어진 미션을 앱을 통해서 달성하세요.

어떤 기능이 있나요? 어떤 기능 원하세요?

앱에 어떤 기능이 들어갈지 궁금하시다면 한국 타이타늄 사용자 모임 부스(B35)로 오셔서 붙어있는 포스트잇을 확인하세요. 원하는 기능을 포스트잇에 적어 주세요. 참신하거나 재치 넘치는 아이디어를 뽑아 또 경품드립니다.
